Detailed Description

Main class of handling quantum assembly. This specific interpreter allows you to interpret open quantum assembly language on the fly. The limit is 15 qubits.

Author
Heiko van der Heijden

void QasmAsyncInterpreter::interpret ( const std::string &  line)

Start to interprets specified open quantum assembly code. This will halt execution until it's done and it will not throw an exception with invalid quantum assembly. If invalid quantum assembly is given, you can get them with the methods, hasErrors and getError
